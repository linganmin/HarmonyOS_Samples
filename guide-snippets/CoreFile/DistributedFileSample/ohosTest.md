# 测试用例归档

| 测试功能                     | 预置条件         | 输入 | 预期输出         | 是否自动 | 测试结果 |
| ---------------------------- | ---------------- | ---- | ---------------- | -------- | -------- |
| 拉起应用                     | 设备正常运行     | \    | 应用被拉起       | 是       | pass     |
| 权限申请                     | 应用被拉起       | \    | 拉起权限申请弹窗 | 是       | pass     |
| 允许多设备协同权限           | 拉起权限申请弹窗 | \    | 选择允许         | 是       | pass     |
| 创建文件并设置文件等级       | 应用被拉起       | \    | 完成文件的创建   | 是       | pass     |
| 将文件从沙箱拷贝到分布式目录 | 设备正常运行     | \    | 文件成功拷贝     | 是       | pass     |
