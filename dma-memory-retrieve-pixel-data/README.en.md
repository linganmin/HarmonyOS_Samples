# DmaMemoryRetrievePixelData

#### Description
本示例介绍了如何利用Node-API实现跨语言交互开发，在DMA_ALLOC模式下，获取内存中存储的准确图像像素数据，确保后期处理图像像素数据时不会因数据不准确导致图像显示异常。
